# Templates

Scheletri riusabili per avviare rapidamente nuovi progetti.

## Template disponibile
- `project-template/`
  - `README.md` (obiettivo, componenti, pinout, compilazione)
  - `src/main.ino` (sketch base)
  - `BOM.md` (lista componenti e costi)
  - `REPORT.md` (report finale del gruppo)

## Come usare il template
1. Copia `templates/project-template/` nella cartella del nuovo progetto.
2. Rinomina la cartella con il nome del progetto.
3. Personalizza i file (`README.md`, `src/main.ino`, `BOM.md`, `REPORT.md`).
